## Part 3. Настройка сети ОС

- Задано имя хоста вида `user-1`.

- Получен IP-адрес от DHCP-сервера, расшифрована аббревиатура DHCP (Dynamic Host Configuration Protocol).
- Определён внешний и внутренний IP-адрес шлюза (gateway).
- Заданы статичные IP, gateway и DNS (1.1.1.1 и 8.8.8.8).

- Проверен пинг удалённых хостов (1.1.1.1 и ya.ru) с 0% потерь.

## Part 5. Использование команды sudo

Пользователь `hallgath` получил права на выполнение команд с помощью `sudo`.  
Истинное назначение `sudo` — временное повышение прав пользователя для выполнения системных команд без постоянного входа под root.  
С помощью `sudo` изменён hostname ОС от имени пользователя `hallgath`.

## Part 9. Установка и использование утилит top, htop
